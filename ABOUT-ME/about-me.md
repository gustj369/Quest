# 나에 대해 — Claude에게 제공하는 맥락

## 기본 정보
- **언어**: 한국어로 소통 (코드 주석도 한국어 우선)
- **개발 경험**: 개인 프로젝트 중심, 바이브코딩 방식 선호
- **작업 환경**: Windows, VSCode, Git

## 이 프로젝트를 만드는 이유
- 스마트워치 없이도 워치 앱에 준하는 접근성을 갖춘 습관 트래커가 필요함
- 매일 반복하는 습관을 RPG 게임의 퀘스트처럼 만들어 동기부여를 높이고 싶음
- 픽셀아트 스타일을 좋아하고, 레트로 게임 감성에 익숙함

## 선호하는 작업 방식
- 큰 변경은 파일 단위로 완전히 교체하는 방식 선호
- 설명보다는 **바로 결과물** 위주로 진행
- 작업 완료 후 스크린샷이나 눈에 보이는 확인을 원함
- 분석 요청 시 구체적인 파일명과 줄 번호까지 포함

## 코드 스타일 선호
- 함수형 컴포넌트 + hooks
- 인라인 스타일과 Tailwind 혼용 (현재 방식 유지)
- 주석은 핵심 로직에만, 너무 많지 않게
- TypeScript 아직 미사용 (추후 도입 예정)
