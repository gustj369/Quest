## ✅ 완료된 작업 (MVP v1)

### 핵심 기능
- [x] 퀘스트 CRUD (생성·조회·삭제)
- [x] 원탭 완료 → 슬라이드 아웃 애니메이션
- [x] 자정 기준 일일 퀘스트 자동 리셋
- [x] repeat 반복 주기 실제 로직 (daily/weekday/weekend/weekly)

### 레벨 & 경험치
- [x] 난이도별 XP (쉬움 10 / 보통 25 / 어려움 50)
- [x] 레벨 공식: 레벨 × 100 XP
- [x] 레벨업 연출 (Framer Motion 골드 플래시)
- [x] XPBar smooth fill (useLayoutEffect로 깜박임 제거)

### UI & UX
- [x] 픽셀아트 디자인 시스템 (CSS 변수 기반)
- [x] 카테고리 탭 + 가로 스크롤 칩
- [x] ALL CLEAR! 배너 (모든 퀘스트 완료 시)
- [x] 빈 상태 메시지 구분 (전체 없음 vs 카테고리 필터)
- [x] QuestCard 애니메이션 타이밍 상수화

### 캐릭터 & 통계
- [x] 픽셀 캐릭터 스프라이트 (레벨에 따라 색상 변화)
- [x] 잔디 달력 (최근 35일 활동 기록)
- [x] 업적 뱃지 (7종, 미획득 흑백 처리)
- [x] 카테고리별 진행도 바 (O(n) 최적화)

### 설정 & PWA
- [x] 설정 바텀 시트 (이름 변경, 알림 토글, 2단계 데이터 리셋)
- [x] PWA 설정 (Service Worker, manifest, 오프라인 동작)
- [x] 홈화면 추가 유도 배너 (beforeinstallprompt, 1회)

## 기술 부채

| 항목 | 내용 | 우선순위 |
|------|------|----------|
| TypeScript 마이그레이션 | 현재 JS, 타입 안전성 부족 | 낮음 |
| 테스트 코드 | 핵심 유틸(xp, date) 단위 테스트 없음 | 중간 |
| 에러 바운더리 | App 수준 React Error Boundary 미설치 | 높음 |
