# 일일 퀘스트

일상의 루틴을 게임처럼 관리하기 위해 만든 일일 퀘스트 실험 프로젝트입니다.<br>
현재는 로컬 웹 환경에서 동작하는 프로토타입이며,<br>
최종 목표는 모바일 환경에 맞춘 루틴 관리 앱으로 확장하는것입니다.

## 만든 이유

해야 할 일을 단순한 체크리스트로 관리한다면 금방 지루해진다고 느꼈습니다.<br>
이 프로젝트는 운동, 공부, 기록 같은 일상 루틴을 퀘스트처럼 완료하고 쌓아가며<br>
작은 성취감을 느끼기 위해 만들었습니다.

## 현재 구현 상태

- 로컬 웹 환경에서 실행
- 일일 퀘스트 목록 확인
- 퀘스트 완료 처리
- 루틴/ 할 일 관리 구조 실험
- 모바일 앱 버전 구현 예정

## 주요 기능

- 일일 퀘스트 목록 관리
- 퀘스트 완료 체크
- 반복 루틴 관리
- 하루 단위 진행 상태 확인
- 자기 관리 습관 형성을 위한 구조 실험

## 사용 목적

- 하루 루틴을 가볍게 관리하기
- 운동, 공부, 기록 같은 반복 행동을 유지해보기
- 체크리스트보다 재미있는 방식으로 습관 만들기
- 모바일로 편의성 확충하기
- 퀘스트 보상으로 보상을 느끼기
